Cache Clear Helper

User Management navigation has been fixed. Please:

  1. Close all browser tabs for this application
  2. Clear your browser cache completely (not just refresh)
  3. Restart your browser
  4. Navigate back to the application
  5. Log in as admin and check the dropdown menu

Browser Cache Clear Instructions:

  • Chrome: Ctrl+Shift+Delete → Clear browsing data
  • Firefox: Ctrl+Shift+Delete → Clear recent history
  • Safari: Cmd+Option+E → Empty caches
  • Edge: Ctrl+Shift+Delete → Clear browsing data
